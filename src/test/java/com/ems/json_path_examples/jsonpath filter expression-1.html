JSON Path Examples

Following are few example jsonpath expressions and results for the above sample json data

JSONPath Expression Description Result
$ Select the root element The output will be same as the json content given as input
$.address.city Select the value of 'city' element which is the direct children of 'address' element "Nara"
$..type Select the value of all the 'type' elements in the input json [
"iPhone",
"home"
]
$.address.length() Select the length of the 'address' element 3
$..* Select all the elements and its value The output will be the list of all the elements and values in the sample input json
$.phoneNumbers[1] Select the 2nd value from 'phoneNumbers' array {
"type": "home",
"number": "0123-4567-8910"
}
$.phoneNumbers[?(@.number)] Select the 'phoneNumbers' element if it have 'number' element within it {
"type": "iPhone",
"number": "0123-4567-8888"
},
{
"type": "home",
"number": "0123-4567-8910"
}
